## INSTRUCTIONS:

1. Keep the 'bot_controller' package and 'final_project' package in the 'src' folder of your catkin workspace
2. Open a terminal with cd as your workspace
3. Use catkin build and source the setup for your workspace
    catkin build
    source ~/<name of your workspace>/devel/setup.bash
4. Run the following command in the same terminal
    roslaunch final_project multiple_robots.launch
5. Open a new terminal and run the following command:
    roslaunch final_project final_project.launch
